# Number Guessing Game in Python

A simple command-line Python game where the computer randomly selects a number between **1 and 100**, and the player has to guess it correctly.

## Overview
This project is a beginner-friendly Python program that demonstrates the use of:
- `random` module
- `while` loops
- `if-elif` conditions
- user input handling
- attempt counting

The game keeps running until the player guesses the correct number. After each guess, it provides feedback:
- **Too low!** if the guessed number is smaller than the target
- **Too high!** if the guessed number is greater than the target

Once the correct number is guessed, the program congratulates the player and displays the total number of attempts.

## Features
- Random number generation between 1 and 100
- Interactive user input
- Hint system for wrong guesses
- Tracks number of attempts
- Simple and easy-to-understand logic

## Technologies Used
- **Python 3**
- **random** module

## How to Run
1. Make sure Python is installed on your system.
2. Copy the code into a Python file, for example: `guess_number.py`
